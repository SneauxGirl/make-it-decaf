Code Switcher landing page featuring background animations, custom themes, and hamburger button/mobile menu. Practice piece for skills building.

Inspo https://github.com/thehashton/codeswitcher/ with style changes.

Didn't like the Prism code themes available so found G Farhad's set of mirror themes (https://github.com/FarhadG/code-mirror-themes).
Tried a few of those - and starred his project for reference - but none were quite right for this build.

Used AI to pull colors from the globals and primary background image on my website and ran through several iterations to adjust for contrast and shades. 
Goal: a cheerful, punchy theme that is also clear and accessible. Kept AI comments on the most recent updates in lieu of prior commits.
